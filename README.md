# MaXnow Portfolio

A premium, reusable portfolio website for MaXnow Technologies built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Folder Structure

- src/app - route layout and main page
- src/components - reusable UI sections such as navbar, footer, and section wrapper
- src/lib - shared content data for services, projects, and stats
- public - static assets (add images and favicons here)

## Run Locally

1. Install dependencies: npm install
2. Start development server: npm run dev
3. Open http://localhost:3000

## Build

npm run build
